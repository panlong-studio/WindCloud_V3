## 标准函数注释格式：

```C
/**        
 * @brief  根据"当前虚拟路径 + 参数"，拼接出服务器上的真实路径
 * @param  res 输出参数，用来保存最终拼好的真实路径
 * @param  size res 缓冲区大小
 * @param  path 当前虚拟路径
 * @param  arg 用户输入的目录名或文件名
 * @return 成功返回 0，失败返回 -1
 */
```

## 开发规范

*   **函数名**：模块名作为前缀 + 动作。
    *   `config_load()`、`log_print()`、`threadpool_add_task()`
*   **变量名**：简洁且有意义。
    *   `int listen_fd;` (不要只用 `int fd;`)
    *   `char *user_name;`
*   **宏与常量**：全大写 + 下划线。
    *   `#define MAX_BUFFER_SIZE 1024`
    *   `#define LOG_LEVEL_DEBUG 0`
*   **结构体**：首字母大写 (PascalCase) 或加 `_t` 后缀。
    *   `typedef struct ThreadPool {...} ThreadPool;`
